Mes Projets

Configuration WLAN sur Packet Tracer

Configuration de points d’accès pour une couverture optimale, sécurisation du réseau avec des protocoles WPA2/WPA3, gestion des VLAN pour segmenter le trafic et tests de connectivité pour garantir un fonctionnement sans faille. Ce projet a renforcé mes compétences en réseaux et l’application de concepts théoriques dans un environnement simulé.

Détails du projet

Implémentation d’une solution WDS

Mise en place d’une solution de déploiement à distance avec Windows Deployment Services (WDS) pour automatiser l’installation de systèmes d’exploitation sur des postes clients, optimisant la gestion des déploiements à grande échelle.

Détails du projet

Migration Windows Server 2019 vers 2022

Migration d’un serveur Windows Server 2019 avec Active Directory vers Windows Server 2022. Étapes : configuration du serveur 2022 comme contrôleur de domaine secondaire, migration des rôles FSMO, transfert des utilisateurs, groupes, GPO et paramètres DNS, suivi de tests pour assurer la continuité de service.

Détails du projet

Mise en place d’un cluster de bascule avec Hyper-V

Configuration d’un cluster de bascule sur Windows Server avec Hyper-V, incluant l’installation de la fonctionnalité de cluster à basculement, la création du cluster et la configuration de la haute disponibilité pour les machines virtuelles, garantissant une infrastructure résiliente.

Détails du projet

Mise en place d’une infrastructure ToIP avec Asterisk et Cisco CME

Configuration d’une solution de téléphonie sur IP avec Asterisk (IPBX open-source) et Cisco CallManager Express (CME). Installation d’Asterisk sur Linux avec support des codecs Opus/MP3 et langue française. Configuration de comptes SIP et plan de numérotation. Gestion des téléphones IP via CME avec protocoles SCCP/SIP et services TFTP.

Détails du projet

Configuration d’un serveur FTP sur Windows Server

Installation et configuration d’un serveur FTP sur Windows Server avec IIS. Création d’un site FTP (`LAMINE-ftp`) avec authentification de base, autorisations de lecture/écriture, et accès via l’adresse IP `192.168.20.66`. Test réussi en interne, permettant le partage et l’upload de fichiers.
